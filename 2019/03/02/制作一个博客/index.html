制作一个博客

关于配置hexo博客（我觉得我是一个踩坑大王） 所以这里有很多解决方法

1、配置环境

(1)下载node.js

more info:下载地址

(2)下载git

more info:下载地址

(3)去github官网创建自己的github账号

more info:官网地址

关于配置github

(1)设置ssh密匙

在windows下查看[c盘->用户->自己的用户名->.ssh]下是否有id_rsa、id_rsa.pub文件，如果没有需要手动生成。
打开gitbush在里面输入
$ ssh-keygen -t rsa -C "youremail@example.com"
注意替换成自己注册的邮箱
然后回车直到出现

就成功了
接下来打开
里面的id_rsa.pub里面的内容复制到注册的github的setting添加ssh key里


(2)创建一个自己的仓库

在自己的github里创建一个仓库

点击new
在这里输入仓库名：（用户名.github.io）

安装并配置hexo
